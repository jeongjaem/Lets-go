Static file notice

이 파일은 이제 메인 앱이 아닙니다.

현재 프로젝트의 실제 프론트엔드는 frontend/의 Next.js 앱이고, 실시간 가격 서버는 backend/의 Express + Socket.IO 서버입니다. 외부 URL에서 localhost 링크를 누르면 사용자 기기 자신의 로컬 주소를 찾기 때문에 실패합니다.

  1. 프론트는 Vercel에 /frontend 루트로 배포합니다.
  2. 백엔드는 Railway 또는 Render에 /backend 루트로 배포합니다.
  3. 프론트 환경 변수 NEXT_PUBLIC_SOCKET_URL에 배포된 백엔드 주소를 넣습니다.
